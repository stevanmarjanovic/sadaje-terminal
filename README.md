# Sada Je Terminal

Text-based 5-minute-step clock in Serbian for terminal use.

![sada je terminal clean](https://github.com/user-attachments/assets/4d3ebf06-1336-4837-b8c2-996c4db22894)

## Usage

```
sadaje [options]
```

|Option            |Description|
|-----------------|-----------|
|`--hide-time`      |**Hide the current time** |
|`--hide-copyright` |**Hide the copyright notice** |
|`--help`           |**Show this help message** |
|`--color=<color>`  |**Set the color of the text**<br>black, darkblue, darkgreen, darkcyan, darkred, darkmagenta, darkyellow, gray, darkgray, blue, green, cyan, red, magenta, yellow, white |
